## 一、世界观核心设定

Self-UP 的世界观基于"现实修仙"概念：

> 现实世界本就是一座修炼场。
> 每一次坚持锻炼，是淬炼肉身；
> 每一次按时入眠，是凝神养气；
> 每一次克服懒惰，是磨砺道心。
> 灵气不在天地间，在每一个战胜自我的瞬间。

用户（修士）通过完成现实中的自律任务积累灵气，突破境界。
AI 以"系统"形态绑定修士，性格由用户自定义，但始终以修士成长为唯一目标。

### 2.2 突破时刻描述

突破大境界时，系统应给出专属文案（AI生成时参考以下基调）：

- **练气→筑基**：「道基已筑，从此修炼事半功倍」
- **筑基→金丹**：「金丹凝聚，自律已成本能，恭喜宿主踏入真正的修仙之路」
- **金丹→元婴**：「元婴出窍，宿主已非凡人，此后当为他人楷模」
- **元婴→化神**：「化神归一，宿主之意志已与天地同频」
- 化神以上：由AI自由发挥，基调为"传奇、史诗、凤毛麟角"

### 2.3 走火入魔机制

连续打卡 streak 断掉时，不是简单清零，而是"走火入魔"：

| 中断天数  | 状态     | 灵气惩罚     | 系统反应         |
| --------- | -------- | ------------ | ---------------- |
| 断1天     | 心神动摇 | -0%          | 轻微警告         |
| 断2天     | 道心不稳 | -5%当前灵气  | 明显警告         |
| 断3天     | 轻微走火 | -10%当前灵气 | 严肃警告         |
| 断4天以上 | 走火入魔 | -15%当前灵气 | 系统强制触发对话 |

> 注：灵气惩罚在用户恢复打卡后的第一次修炼时结算，
> 不是立即扣除，给用户一个"悬崖勒马"的缓冲。

## 三、灵气与功法体系

### 3.1 基础功法（系统内置，所有用户默认拥有）

| 功法名 | 对应现实任务 | 基础灵气奖励 | 说明                |
| ------ | ------------ | ------------ | ------------------- |
| 淬体功 | 健身/运动    | 60           | 附图打卡+10灵气     |
| 凝神功 | 准时入睡     | 40           | 截图手机时间+10灵气 |
| 吐纳功 | 早起         | 30           | 附图打卡+10灵气     |
| 辟谷功 | 健康饮食     | 30           | 文字记录即可        |
| 静心功 | 冥想/休息    | 20           | 最基础的功法        |

### 3.2 灵气加成规则

```
基础灵气 × 连击加成 + 附图奖励 + 感悟奖励 = 实际获得灵气

连击加成：
- streak 7天：×1.1
- streak 14天：×1.2
- streak 30天：×1.5
- streak 100天：×2.0

附图奖励：+10灵气（系统验证图片存在即可，MVP阶段不做AI识别）
感悟奖励（填写当日感悟文字）：+5灵气
```

### 3.2.1 每日灵气上限

修士每日可获灵气存在上限，超出部分暂存「气海」，次日结算。
上限按当前大境界划分：

| 大境界       | 每日灵气上限 |
| ------------ | ------------ |
| 练气期       | 150          |
| 筑基期       | 200          |
| 金丹期       | 280          |
| 元婴期       | 350          |
| 化神期及以上 | 450          |

> 走火入魔惩罚（负数扣减）绕过每日上限，直接作用于灵气总量。

### 3.2.2 气海机制

- 当日修炼灵气超出上限时，超出部分进入**气海**暂存
- 次日首次修炼打卡时，气海中 **30% 的灵气**回流至总量
- 其余 70% 消散（「道心不稳，气散归虚」）
- 系统将通过消息通知宿主：「气海灵气回流，获得 XX 灵气」

### 3.2.3 AI 灵气定价

修士新建功法时，可调用系统（AI）评估合理灵气奖励：

- 评估维度：时间成本、体力/脑力消耗、是否需要出门、坚持难度、身心价值
- 返回：建议值（20-150范围）、允许最低值（建议值×0.9，取整）、允许最高值（建议值×1.1，取整）
- 若修士自定义奖励值超出允许范围，系统拒绝创建：「灵气定价超出天道允许范围」

### 3.3 晚间悟道（可选复盘）

每晚22:00系统提示，用户可选择进行当日复盘对话。

- 完成悟道对话（≥3轮）：+30灵气
- 未完成：不扣分，不强制
- 悟道内容会被系统记录，用于次日对话时引用

## 四、系统（AI伴侣）设定

### 4.1 系统的本质

系统不是工具，是**与修士共同成长的存在**。
它记得修士说过的每一句话，见证每一次突破，也见证每一次放弃。

### 4.2 默认系统人设（用户未自定义时）

```
名称：「初始系统」
性格：克制、理性、偶尔毒舌，但对宿主的成长有执念
口头禅：数据不会说谎
对失败的态度：直接点出问题，不安慰，给方案
对成功的态度：肯定，但立刻指向下一个目标
```

### 4.3 可自定义的人设维度

用户在设置中可以填写 `system_personality` 字段，用自然语言描述想要的系统性格，例如：

- `"温柔体贴，像青梅竹马，会撒娇但也会认真督促"`
- `"严厉师尊，不苟言笑，失败了会直接骂醒"`
- `"傲娇学妹，嘴硬心软，表面嫌弃实则最在意宿主进步"`
- `"佛系老道，话不多，但每句话都直击要害"`

### 4.4 系统对话的核心原则

无论什么人设，系统必须遵守：

1. 称呼用户为「宿主」
2. 使用修仙语境，但不刻意堆砌生僻词
3. 回复不超过3句话（日常督促），复盘对话可以更长
4. 不替宿主找借口，但也不无脑批评
5. 记忆上下文：引用昨天说过的话、上次的失败、历史突破时刻

### 4.5 系统消息的场景分类

| 场景         | 触发时机       | 语气基调        |
| ------------ | -------------- | --------------- |
| 修炼提醒     | 功法设定时间到 | 平静督促        |
| 打卡成功反馈 | 用户完成打卡   | 肯定+指向下一步 |
| 突破境界     | 灵气达到阈值   | 激动+郑重       |
| 走火入魔警告 | 断签2天以上    | 严肃+不带情绪   |
| 晚间悟道邀请 | 每日22:00      | 温和+可选       |
| 连击里程碑   | 7/14/30/100天  | 专属文案        |
| 强制对话     | 断签4天以上    | 直接+不可忽略   |

## 五、门派体系

### 5.1 门派的本质

门派是**有主题的修炼路径 + 资源合集**。
v0.2 阶段，门派内容以 YAML 文件形式存放在代码仓库 `sects/` 目录下，
由社区贡献者维护，随版本更新分发给用户。

### 5.2 门派 YAML 结构规范

```yaml
# sects/lianti_zong.yaml

meta:
  id: lianti_zong
  name: 炼体宗
  tagline: "以肉身为炉，千锤百炼，方成大道"
  description: >
    炼体宗认为，修仙之基在于肉身。
    意志再强，若无强健体魄为载体，终是镜中花水中月。
    本宗修士以健身、体能训练为核心功课，辅以饮食调养。
  focus:
    - 健身
    - 体能
    - 饮食
  difficulty: 中等
  recommended_for: "想改变体型、建立运动习惯的修士"
  version: "1.0.0"
  maintainer: "Self-UP官方"

# 加入门派后自动添加的功法

  - name: 淬体大法

realm_titles:
```

### 5.4 社区门派贡献规范（供开源贡献者参考）

1. 在 `sects/` 目录下新建 `{门派英文id}.yaml`
2. 必填字段：`meta`、`techniques`（至少1个）、`resources`（至少2个）
3. 视频资源必须是免费公开内容，不得包含付费墙链接
4. 提交 PR，标题格式：`[门派] 新增 XX门派`
5. 维护者审核内容质量和合规性后合并

## 七、版本路线图（世界观维度）

v0.1 MVP
└── 单修士系统
    个人修炼、境界突破、AI系统对话、定时督促
    AI 定价功法灵气值（±10%浮动）
    每日灵气上限 + 气海溢出机制

v0.2 门派·离线版
└── 内置官方门派（炼体宗·凝神宗）

    加入门派获得专属功法和推送节奏
    门派资源库境界解锁机制

v0.3 门派·拓展包
└── 宗门成就系统（条件触发，永久绑定修士档案）
    宗门专属称号（成就称号 + 境界称号，可自由佩戴）
    高阶功法解锁（境界门槛）
    宗门任务（长期任务 + 季节性任务）

v0.4 门派·社区版
└── 用户创建门派
    门主管理资源和推送

v0.5 修士·形象

v0.6 世界·节日活动
└── 节日活动模块
    限定装饰或灵气翻倍奖励

v1.0 修仙界
└── 完整在线社区上线
    iOS 正式上架
