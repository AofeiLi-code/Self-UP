# Self-UP · 修仙自律系统

> 把每一次自律当作一次修炼，突破自我即是突破境界。

Self-UP 是一款以修仙世界观包装的自律打卡应用。完成现实任务（健身、早睡、读书…）即可获得「灵气」，积累灵气突破境界，随身 AI 系统全程陪伴督促。

## 功能概览

### 修炼系统

- **每日打卡**：选择功法（对应现实任务），完成后记录修炼，获得灵气奖励
- **附图 + 感悟**：打卡时可上传修炼凭证照片（+10 灵气）或填写心得感悟（+5 灵气）
- **连续修炼加成**：连续打卡 7 / 14 / 30 / 100 天，灵气倍率逐级提升（最高 ×2.0）
- **境界突破**：灵气累积达到阈值自动突破，练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 合体期 → 大乘期 → 渡劫期，共 8 大境界 × 4 阶
- **每日灵气上限**：按境界设定每日上限（150–450 灵气），避免刷分
- **气海机制**：当日超出上限的灵气流入气海暂存，次日回流 30%
- **走火入魔惩罚**：断修后复返，根据断修天数扣减当前总灵气（最高 -15%）

### 功法管理

- **自定义功法**：可自由添加任意自律任务，设定名称、现实描述和定时修炼时刻
- **AI 天道定价**：调用 AI 为功法灵气奖励给出合理建议值（±10% 浮动区间）
- **废弃 / 恢复**：可停修任意功法，也可一键恢复；支持批量清空已废弃功法

### AI 随身系统

- **打卡反馈**：每次修炼后由 AI 生成个性化系统反馈，文风根据境界和连击天数自动调整
- **自由对话**：随时与随身系统对话，获得修炼指引或日常陪伴
- **人设自定义**：可自定义系统称号和人设口吻（严厉督促 / 温柔鼓励 / 神秘冷淡等）
- **定时督促**：设定修炼时刻后，系统会在指定时间发送督修消息

### 系统消息

- **消息中心**：系统自动推送境界突破贺报、气海回流通知、宗门任务完成提醒
- **单条删除 + 一键清空**：可逐条删除或一键清空全部消息历史

### 宗门系统

- **正式拜入**：每位修士只能正式拜入一个宗门，自动获得该宗门的专属功法
- **游历制度**：可同时游历多个宗门，选择性添加游历宗门的功法至每日功课
- **官方宗门**（共 2 个，持续拓展）：
  - **炼体宗** — 健身·体能·饮食方向
  - **凝神宗** — 早睡·作息·规律生活方向
- **宗门秘籍**：各宗门内置学习资料库，按修士当前境界标注推荐程度
- **宗门任务**：长期修炼目标（累计天数 / 连续天数），完成后领取灵气奖励

### 修炼历史

- **历史记录**：按日期分组查看全部打卡记录，显示功法名、灵气奖励、时间
- **照片浏览**：缩略图展示修炼凭证，点击全屏灯箱查看原图
- **感悟摘要**：打卡时填写的心得在历史页完整展示

### 账号系统

- 注册 / 登录（JWT 鉴权）
- 深色 / 亮色主题自由切换
- 全站移动端适配，支持添加到手机主屏幕使用

## 快速开始

### 环境要求

- Python 3.11+
- Node.js 20+
- ngrok（可选，用于手机访问）

### 1. 克隆项目

```bash
git clone https://github.com/你的用户名/Self-UP.git
cd Self-UP
```

### 2. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env，填入你的 DeepSeek API Key 和 JWT 密钥
```

### 3. 启动后端

```bash
cd backend
python -m venv .venv

# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 4. 启动前端

新开一个终端：

```bash
cd frontend
npm install
npm run dev
```

### 5. 访问

浏览器打开 [http://localhost:5173](http://localhost:5173)

> 注：前端 dev server 通过 Vite proxy 将 `/api/*` 转发至 `localhost:8000`，无需额外配置跨域。

### Docker（可选）

```bash
docker-compose up --build
```

后端监听 `:8000`，前端监听 `:5173`。

### 手机访问

1. 确保本地服务已启动
2. 运行 `ngrok http 5173`，获取公网地址
3. 用手机浏览器打开 ngrok 提供的 `https://` 地址
4. 点击浏览器菜单「添加到主屏幕」获得类 App 体验

## 环境变量说明

参考 `.env.example`：

| 变量               | 用途                                | 获取方式                                               |
| ------------------ | ----------------------------------- | ------------------------------------------------------ |
| `DEEPSEEK_API_KEY` | DeepSeek API 密钥，所有 AI 功能必需 | [platform.deepseek.com](https://platform.deepseek.com) |
| `JWT_SECRET_KEY`   | JWT 签名密钥，用于用户登录鉴权      | 任意随机字符串（生产环境请使用长随机串）               |
| `DATABASE_URL`     | 数据库路径（可选）                  | 默认 `sqlite:///./selfup.db`，留空即可                 |

## 世界观

Self-UP 将现实自律比喻为修仙之路——每一次健身是「淬体」，每一次早睡是「凝神」，每一天的坚持都是修为的积累。随身 AI 系统扮演陪伴修士的「系统」角色，以修仙口吻给出反馈与督促。
